# Product Catalog Admin Panel

This is a simple **Product Catalog Admin Panel** built with **Angular 20**, **TailwindCSS**, and **PrimeNG**. The project allows you to view, filter, search, paginate, and edit products.

## Features

- Dashboard with product statistics
- Product List Page
  - Responsive grid layout
  - Search by product title
  - Filter by category
  - Pagination (5, 10, 20 items per page)
  - Loading state handling
- Product Detail Page
  - View product details
  - Edit product title, price, category, description, and image
- Navbar shows current page (Dashboard / Products / Product Detail / Login)
- Error page for 404

## Technologies Used

- **Angular 20** (standalone components, signals, and effects)
- **TailwindCSS** for styling
- **PrimeNG** for paginator component
- **Fake Store API** for product data
- **Vite** for fast development server
